Author Guide: Adding books + formatting Google Docs

Edit the BOOK_LIBRARY block in this file. Duplicate one entry to add another tile/book.

  1. Set id and title.
  2. Paste your doc URL in chaptersGoogleDoc.
  3. In Google Docs, put each chapter heading on its own line as either ## Chapter 1: Name or Chapter 1: Name.
  4. Everything until the next chapter heading becomes that chapter's page text.

If the Doc is private, visitors will see fallbackChapters content until you make the Doc visible again.
